To check if the series $\sum_{n=1}^{\infty}a_{n}$ is divergent, apply the following tests, if one fails, move to the next.
$$
\begin{gather}
1.\ \lim_{ n \to \infty } a_{n}\neq 0 \\
2.\ a_{n}=a_{0}r^{n}\implies |r|>1 \\
3.\ a_{n}=\frac{1}{n^{p}}\implies p\leq 1 \\
4.\ \sum_{n=1}^{\infty} b_{n}\text{ is divergent, }a_{n}\leq b_{n}\implies a_{n}\ \text{is divergent} \\
5.\ a_{n},b_{n}\geq 0\ \text{and}\ \lim_{ n \to \infty } \bigg| \frac{a_{n}}{b_{n}}\bigg|> 0\implies \text{Both either diverge or converge} \\
6.\ a_{n}=(-1)^{n}b_{n}\ \text{converges if }\lim_{ n \to \infty } b_{n}=0\ \text{and}\ b_{n+1}\leq b_{n}\ \forall\ n\geq 1 \\
7.\ \text{Absolute covergence}\implies \text{convergence} \\
8. \ \lim_{ n \to \infty } \bigg|\frac{a_{n+1}}{a_{n}}\bigg|<1\implies \text{Absolute convergence} \\
9.\ \lim_{ n \to \infty } \sqrt[n]{ |a_{n}| }<1\implies \text{Absolute convergence} \\
10.\ a_{n}=f(n),\ f(n)>0,f'(n)<0,a_{n}\ \text{converges if} \int_{1}^{\infty} f(x) \, dx \ \text{converges}
\end{gather}
$$
