There are 20 common amino acids. A four letter code in pairs of two (AUGC) was insufficient, since only 16 iterations could be made.
However, a code in groups of three had 64 iterations and was sufficient.
So, a code for translation of nucleotide sequence in RNA was developed to translate it into amino acids.
